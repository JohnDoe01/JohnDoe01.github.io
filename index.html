Supplying print ready material

Colour

  • All material must be supplied in CMYK
  • PMS and spot colours must be converted to process colours
  • Embeded colour profiles should not be used
  • No colour patterns, duotones or tritones
  • Total ink weight should not exceed 310% with a 90% black maximum
  • 14-18% dot gain allowance

Screen

  • FOUR COLOUR - 133 lines per inch
  • MONOTONE - 110 lines per inch

Images

  • CMYK
  • EPS or TIFF format
  • Image minimum of 300dpi at 100% print size
  • Black and white line art minimum 2,400dpi at 100% print size
  • All transparencies must be flattened

Type

  • Postscript fonts or outline fonts should be used
  • TrueType and CID fonts not accepted
  • All fonts should be embedded
  • Type should be 5mm within the trim size
  • Any type less than 10pt bold is not suitable for reverse printing
  • It is reccomended that all black type be made using process black only

Solid background

  • Large solid black areas should have an additional 30% cyan tint beneath to provide additional density

PDF

  • High resolution PDF to be supplied

Proofs

  • For accurate colour reproduction a full size 3DAPv3 Papertype 3 proof must be provided. For details on how to create a 3DAPv3 proof and approved proofing systems, visit www.3dap.com.au
